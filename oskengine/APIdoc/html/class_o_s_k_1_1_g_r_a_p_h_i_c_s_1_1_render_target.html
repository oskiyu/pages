void RenderTarget::Create ( const Vector2ui &  targetSize,
DynamicArray< RenderTargetAttachmentInfo >  colorInfos,
RenderTargetAttachmentInfo  depthInfo 
)

Crea el render target con la información dada, para su uso como render target en shaders de fragmentos.

Parámetros
targetSizeResolución, en píxeles.
colorInfosInformación de las imágenes finales de color.
depthInfoInformación de la imagen de profundidad.
Precondición
Debe haber información para al menos una imagen de color.
Postcondición
El render target podrá usarse como renderpass para shaders de fragmentos.
Nota
Pueden incluirse varias imágenes de color objetivo.
Cada imagen objetivo de color es independiente y puede tener formatos y usos distintos al resto de imágenes objetivo.
Excepciones
InvalidArgumentExceptionSi no se cumple la primera precondición.

◆ Resize()

void RenderTarget::Resize ( const Vector2ui &  targetSize)

Cambia de tamaño las imágenes de renderizado, tanto las de color como las de profundidad.

Parámetros
targetSizeNueva resolución de las imágenes de renderizado.
Precondición
El render target debe haber sido correctamente creado mediante RenderTarget::Create o RenderTarget::CreateAsFinal.
Atención
Esta función destruye las imágenes anteriores, por lo que si una instancia de material referenciaba estas imágenes, se deberá actualizar con las nuevas imágenes.
